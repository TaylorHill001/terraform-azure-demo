## Goal
Understand :
- HCL function in Terraform

## Setup
Login to Azure:
```bash
az login
```

Initialize your workspace once:
```bash
# Init cloud provider plugin & template plugin
terraform init
```

## Usage
Apply deployment:
```bash
terraform apply
```

## Teardown
Destroy resources when finished:
```bash
terraform destroy
```
